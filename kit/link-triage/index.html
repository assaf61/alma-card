link-triage — התחל כאן

שי, זה הכלי שכבר הרצנו עליו יחד, עכשיו ארוז שיתקן את עצמו.

זו גרסה חדשה (2), שמתקנת את מה שהיה מסובך בהתקנה הקודמת. עכשיו זה מתקין את עצמו.

מה זה עושה

link-triage

נותנים לו לינק אחד וכוונה חופשית ("בדוק לי, האם כדאי, מצא לי"), והוא מאמת מול העולם, נותן חוות דעת ישרה בלי להחניף, מסדר את הלוגיסטיקה עד סף הבלתי-הפיך, ומגיש דוח וטיוטות מוכנות. לעולם לא שולח ולא מזמין בשמך.

להפעלה, כותבים לעוזר: בדוק לי את הלינק הזה, האם כדאי לי, ומצא לי גם לינה באזור: <לינק>

בצ'אט רגיל: בצ'אט זה יעבוד בגרסה מוקטנת: יקרא את הלינק וייתן חוות דעת. את המחקר הרחב והטיוטות המוכנות מקבלים רק עם כלי סוכן.

קודם, השג את הכלי

הכי מהיר: אם יש לך Claude Code או סוכן שמריץ פקודות, העתק את השורה והדבק לסוכן שלך. הוא יוריד, יחלץ ויתקין הכל לבד.

הורד את הקובץ מ-https://assaf61.github.io/alma-card/kit/link-triage.zip , חלץ אותו לתיקייה, וקרא את START.md שבפנים ובצע בדיוק מה שכתוב בו כדי להתקין את הכלי.
או הורד ידנית (ZIP)

אחרי ההורדה, המשך לפי "איך מתקינים" למטה, לפי מה שיש לך.

איך מתקינים — לפי מה שיש לך

מצב 1

יש לי כלי סוכן (Claude Code / Codex / Cursor / Gemini CLI)

  • הכי מהיר, ווינדוס עם Claude Code: לחיצה כפולה על install.cmd מתקינה את הכלי לבד, ואומרת מה לכתוב.
  • בכל סוכן אחר: גוררים לתוכו את START.md ואומרים "עקוב אחרי זה". הוא יזהה את עצמו, יתקין, וידווח.
מצב 2

אני בצ'אט רגיל (ChatGPT / Gemini / Claude בדפדפן)

פותחים את SKILL.md של הכלי, מעתיקים הכל, מדביקים בצ'אט וכותבים "עבוד לפי זה". ראו למעלה מה כל כלי עושה בצ'אט.

מצב 3

אין לי כלום, או לא בטוח מה עדיף

פותחים את 'רוצה להבין יותר' למטה: מה ההבדל בין הסוגים ואיך עוברים לגרסה שבאמת עושה דברים.

רוצה להבין יותר — סוגי העוזרים, ומה עדיף

העוזרים נחלקים לפי יכולת, לא לפי מותג. ההבדל האמיתי: האם העוזר יכול להריץ תוכנות ולגעת בקבצים על המחשב, או רק לדבר.

סוגדוגמאותמה הוא יכול
סוכן בטרמינלClaude Code, Codex, Gemini CLI, Cursor, Clineמריץ תוכנות, כותב קבצים, מתקן את עצמו. פה הכלי עובד במלואו.
צ'אט עם ארגז-חולChatGPT עם code interpreterמריץ קוד בענן שלו על קבצים שמעלים, אבל לא נוגע במחשב.
צ'אט רגילChatGPT, Gemini, Claude בדפדפןחושב ומדבר מצוין, לא מריץ שום דבר על המחשב.

איך עוברים לגרסה החזקה: קודם מנסים בצ'אט חינם, בלי להתקין כלום, רק להרגיש. אם אהבתם ורציתם שזה גם יעשה דברים בפועל, מתקינים סוכן בטרמינל (Claude Code הוא נקודת פתיחה טובה). מאותו רגע הכלי הזה, וכל כלי עתידי, פשוט יתקינו את עצמם.

עלות, בגדול: לרוב העוזרים יש שכבה חינמית לנסות, ושכבה בתשלום (בדרך כלל עשרות שקלים בודדות לחודש) למודלים החזקים ולשימוש כבד. המספרים זזים, שווה לבדוק עדכני לפני שמשלמים.

אחרי שניסית — שאלון קצר

דקה, כמה שאלות עם תשובות ללחיצה ומקום להוסיף במילים שלך. השם והכלי כבר ממולאים מראש עבורך. יש בו גם שאלה חדשה: האם הגרסה הזו טובה יותר מהקודמת. לחיצה אחת פותחת אותו, ומשם זה נאסף לבד. אין מה להעתיק או לשלוח.

פתח את השאלון נפתח בלשונית חדשה, נשמר לבד בסיום.
